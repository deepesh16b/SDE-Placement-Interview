### #31. Sliding Window Maximum

```
Input: nums = [1,3,-1,-3,5,3,6,7], k = 3
Output: [3,3,5,5,6,7]
Explanation: 
Window position                Max
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```
Problem: https://leetcode.com/problems/sliding-window-maximum/


```cpp
vector<int> maxSlidingWindow(vector<int>& nums, int k) {
    int n = nums.size();
    deque<int> q;
    vector<int> ans;

    for (int i = 0; i < n; ++i) {
        if (!q.empty() && i >= k && q.front() == nums[i - k])
            q.pop_front();
        while (!q.empty() && q.back() < nums[i])
            q.pop_back();
        q.push_back(nums[i]);
        if (i + 1 >= k)
            ans.push_back(q.front());
    }

    return ans;
}
```
