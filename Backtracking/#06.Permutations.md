# #6. Permutations

Find the permutations of string inputted.

```
Input: ABC
Output: ABC ACB BAC BCA CBA CAB
```

Steps:
- At each level, we get new character to swap and add to the level.

**CPP - Backtracking**
```cpp
void permute(string s, int level = 0) {
    if (level == s.size()) {
        cout << s << endl;
        return;
    }

    for (int i = level; i < s.size(); i++) {
        swap(s[i], s[level]);
        permute(s, level + 1);
        swap(s[i], s[level]);
    }
}
```
