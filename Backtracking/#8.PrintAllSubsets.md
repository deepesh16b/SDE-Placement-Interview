### #8. All Subsets

```
Input: ABC
Output: A B AB C AC BC ABC
```

**Using Bitmask**
```cpp
void allSubsets(string s) {
    int n = s.size();
    for (int mask = 0; mask < 1 << n; ++mask) {
        string sub = "";
        for (int j = 0; j < n; ++j) {
            if (mask & (1 << j)) {
                sub += s[j];
            }
        }
        cout << sub << endl;
    }
}
```

**Using Backtracking**
```cpp
void allSubsets(string s, int n, int pos, string ans = "") {
    if (pos >= n) {
        cout << ans << endl;
        return;
    }

    allSubsets(s, n, pos + 1, ans);
    allSubsets(s, n, pos + 1, ans + s[pos]);
}
```
